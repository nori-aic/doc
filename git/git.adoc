=== 初期設定

1.Git Bashを起動する。 +
image:images/git_0018.png[] +

image:images/git_0019.png[]

2.GitHubのユーザ名 or 他人から見て区別できるユーザ名を設定する。 +
----
$ git config --global user.name "JohnSmith"
----

3.メールアドレスを設定する。 +
----
$ git config --global user.email "john.smith@email.com"
----

4.設定内容を確認する。 +
----
$ git config --list
----
